# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, education, and contact information in a clean and responsive web interface.

## 📖 About

This portfolio was developed to present my work, technical skills, and projects in one place. It serves as an online resume where visitors can learn about my background, explore my projects, and connect with me.

## ✨ Features

* Responsive design for desktop and mobile devices
* Professional landing page
* About Me section
* Skills showcase
* Project portfolio
* Education details
* Contact section
* Resume download
* Social media links

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/janathiru25/portfolio.git
```

### Open the project

```bash
cd portfolio
```

Open the `index.html` file in your browser or use a local development server such as the Live Server extension in Visual Studio Code.
